## Approach

Three stages, documented in `advanced_solver.py`:

**Stage 1 — Edge signal**
Extract satellite RGB patch, run Sobel edge detection (primary signal).
Blend 70/30 with boundaries.tif hints.
Gate on satellite edge strength alone so boundary hints cannot dilute
a strong satellite signal.

**Stage 2 — Shift via FFT cross-correlation**
Render the official plot outline as a Sobel edge mask.
Cross-correlate with the blended edge map inside a ±50 m search window.
Gaussian blur both before correlating to tolerate sub-pixel errors.
Normalise score_margin by patch energy so small plots don't inflate it.

**Stage 3 — Decision and confidence**

Hard gates (always flag):
- No recorded area → road / public land
- Area ratio outside [0.60, 1.70] → shape or history error
- Satellite edge mean < 0.10 → nothing to align to

Confidence = 0.50 × score_margin + 0.30 × ratio_fit + 0.20 × hint_density

If per-plot confidence < 0.55, try village-wide median shift as fallback.
Fallback confidence capped at 0.50 — never overrides a strong per-plot signal.
If confidence still < 0.55 after fallback → flagged.

**Design philosophy:** flag when evidence is weak. Each threshold has a named
reason in the code. The goal is a method that holds up on data it has never
seen, not one tuned to the 6 public truth plots.
